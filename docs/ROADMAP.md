# Roadmap

This roadmap is intentionally small: a practical snapshot of where Tunet is headed, not a promise list.

## Now

- Keep idle performance and bundle size under control.
- Smooth the everyday dashboard editing flow.
- Make connection, auth, and profile sync behavior easier to trust.

## Next

- Improve mobile editing and card organization.
- Add clearer guidance for configuring advanced card behavior.
- Continue tightening test coverage around auth, drag-and-drop, and modal workflows.

## Recently Improved

- Reduced idle dashboard CPU usage from continuous background motion.
- Fixed the `ip-address` Dependabot advisory through `express-rate-limit` updates.
