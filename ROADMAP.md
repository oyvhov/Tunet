# Roadmap

This document outlines the planned features and improvements for Tunet.

## Current Version: 1.6.1

With version 1.6.x, we've delivered key security and usability enhancements:
- ✅ **Settings Lock**: PIN protection for edit mode and settings
- ✅ **Enhanced Person Card**: Redesigned popup with detailed battery/telemetry and dynamic map
- ✅ **Enhanced Car Card**: Improved sensor mapping and status visualization
- ✅ **Unit System**: Full support for Imperial/Metric units across all cards
- ✅ **Mobile UX**: Better touch interactions and pinch-zoom support
