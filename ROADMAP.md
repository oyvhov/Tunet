# Roadmap

This document outlines the planned features and improvements for Tunet.

## Current Version: 1.1.0

With version 1.1.0, we've established a solid foundation:
- ✅ Synchronized versioning between application and addon
- ✅ Graduated from beta to stable release
- ✅ Automated release process with version validation

## Upcoming Features

### Version 1.2.0 (Q1 2026)
- [ ] Enhanced dashboard customization options
- [ ] Additional Home Assistant entity integrations
- [ ] Performance optimizations for large installations
- [ ] Improved mobile responsiveness

### Version 1.3.0 (Q2 2026)
- [ ] Theme customization and dark mode improvements
- [ ] Widget marketplace/library
- [ ] Multi-language support expansion
- [ ] Accessibility improvements (WCAG 2.1 AA compliance)

### Version 2.0.0 (Q3 2026)
- [ ] Plugin architecture for community extensions
- [ ] Advanced automation workflows
- [ ] Real-time collaboration features
- [ ] Cloud backup and sync capabilities

## Long-term Vision

- **Extensibility**: Build a robust plugin ecosystem
- **Performance**: Sub-second load times for all dashboard views
- **Accessibility**: Full WCAG 2.1 AAA compliance
- **Community**: Foster an active community of contributors and users
